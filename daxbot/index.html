Commands for Non-Mods:

db!commands = Pulls up these commands

db!youtube = Gives a link to my YouTube Channel

db!twitter = Gives a link to my Twitter db!twitch = Gives a link to my Twitch

db!instagram = Gives my Instagram name

db!snapchat = Gives my Snapchat name

db!insult [@user] = DM's an "insult" to mentioned user

db!idle-on/db!idle-off Sets status to idle on/off

db!dnd-on/db!dnd-off Sets status to DND on/off

Commands for Mods:

db!ban @[user] = Bans the mentioned user.

db!kick @[user] = Kicks the mentioned user.

Commands for DMs:

db!something = Something.

db!dmcommands = Shows DM Commands
